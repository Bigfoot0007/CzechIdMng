# CzechIdMng

2nd version of [CzechIdM](http://www.czechidm.com/) will be based on technologies:
* Backend (java):
  * jdbc db (primary PostgreSQL)
  * Hibernate ORM
  * Spring (data, data rest, hateoas, security, integration)
  * Activiti (workflow)

* Frontend (javascript)
  * React
  * Redux

# CzechIdM 7

The seventh version of [CzechIdM](http://www.czechidm.com/) will be based on technologies:
* Backend (java):
  * jdbc db (primary PostgreSQL)
  * Hibernate ORM
  * Spring (data, hateoas, security ...)
  * Activiti (workflow)

* Frontend (javascript)
  * React
  * Redux
